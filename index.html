Tri-color Palette Generator

This tool generates tri-colour palette where one colour is base color, and other two are complementary colors separated by 60` angle on the color wheel. Last three colors are corresponding complementary colors. (code)

Use this palette as a way to find good color combination, then use this amazing tool to choose shades and add little randomness to make it your perfect web color palette.
